### 🏗️ Project Highlights

> **🤖 RunwayIQ** — AI Financial Intelligence Platform *(GenAI Genesis 2026)*
> 
> Full-stack AI platform with Node.js, Express, React, Supabase + Prisma. Integrated **Claude AI API** for automated CFO-level reports with burn rate analysis, runway forecasting, and risk scoring. Built dashboards with Recharts.

> **🗺️ SenseMap** — AI Accessibility Mapping Platform *(HackCanada 2026)*
> 
> NLP system converting unstructured reviews into structured sensory insights (noise, lighting, crowds). Interactive map with personalized filtering for sensory-sensitive and autistic users.

> **🏙️ UrbanSight** — Smart City Reporting Platform *(GDG McMaster — 🏆 Runner-Up)*
> 
> Full-stack civic reporting app with Firebase real-time sync, Auth, and Cloud Storage. Shipped as **PWA** on iOS, Android, and web. UN SDG 11 Smart Cities Challenge.

> **✉️ ReplyWise** — AI Email Automation System *(HackHive)*
> 
> AI Gmail assistant using Angular, Node.js, and LLM API integration. Chrome extension for contextual reply generation with async processing.
